# 项目文档
## 项目需求
建立一个工行的积分兑换系统的ai助理。工行的积分兑换可以直接在工行积分商城用积分买东西，也可以兑换京东e卡来然后去京东买东西。
需要ai助理工作如下：
如用户输入想买的东西，或者表达了某个方向如给父母礼物等。
ai助理根据工行的商品目录看是否有合适的东西。如果有，就根据工行的商品价格和京东api得到价格看是直接在工行商城买还是在换京东卡去京东买合适--如果京东的合适就给出连接。如果工行商品没有合适的，就去京东看价格，并且给出连接。
如果用户的积分不够，就需要根据RAG数据库去将最合适的攒积分的方法列出来。

用户可能开始的需求不够明确，需要agent不断和用户交互澄清确切的需求。


## 如何运行单元测试

要运行项目中的单元测试，请使用以下命令：

```bash
python -m unittest discover -s tests
```

或者指定特定的测试文件：

```bash
python -m unittest tests.test_hello
```

这将执行 `tests/` 目录下的所有测试文件。
